# Glossary

## Terms

**Market**: A prediction market where users can stake on outcomes

**Stake**: Amount of STX tokens placed on an outcome

**Outcome**: Result of a market (YES or NO)

**Resolution**: Process of determining market outcome

**Oracle**: Entity authorized to resolve markets

**Governance**: System for proposing and voting on changes

**Proposal**: Suggested change to the platform

**Principal**: Stacks address

**Contract Call**: Transaction that executes contract function

**Read-Only Function**: Function that doesn't modify state

**Public Function**: Function that modifies state

**Post Condition**: Safety check for transactions

**Anchor Mode**: Transaction confirmation mode

**Microstack**: Smallest unit of STX (1 STX = 1,000,000 microstack)

**Block Height**: Number of blocks in the blockchain

**Transaction ID**: Unique identifier for a transaction

**Nonce**: Transaction sequence number

**Fee**: Cost to execute a transaction

**Gas**: Computational cost of operations
